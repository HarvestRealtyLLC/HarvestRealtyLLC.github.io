üìß PurelyMail Email Setup for New Agents

Follow these steps to create a new email account for an agent:

Step 1: Login to PurelyMail Admin

  1. Go to https://purelymail.com/manage/login
  2. Login with admin account: HarvestRealtyLLC@purelymail.com
  3. Enter the admin password

Step 2: Navigate to Users

  1. Click on "Users" in the top navigation menu
  2. You will see a list of all current email users

Step 3: Create New User

  1. Click "Create new user" button
  2. In the Name field, enter the agent's first name (lowercase)
    Example: for "John Smith" enter "john"
  3. IMPORTANT: Select domain "harvestrealty.llc" from the dropdown
  4. Generate a random password or create one (minimum 12 characters recommended)
  5. Check "Enable search indexing" ‚úì
  6. Check "Enable spam filtering" ‚úì
  7. Check "Enable password reset" ‚úì
  8. Click "Save"

Step 4: Send Credentials to Agent

  1. Send the new agent their login credentials:
  2. Email: firstname@harvestrealty.llc
  3. Password: (the generated password)
  4. Webmail: https://purelymail.com/webmail
  5. Direct them to the Onboarding Page for full setup instructions
